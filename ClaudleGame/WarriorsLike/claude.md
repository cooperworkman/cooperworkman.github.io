# CLAUDLE - Warriors-like 無雙遊戲

## 概述
一款以無雙割草為核心的第三人稱動作遊戲，擁有 1600 隻史萊姆、連擊系統、三階角色進化系統、魔王戰、程序化音效，以及強烈的打擊感回饋。使用純網頁技術（HTML + Three.js r128）打造，單一 HTML 檔案即可在瀏覽器中遊玩。

---

## 技術架構

| 項目 | 技術選型 |
|------|---------|
| 3D 引擎 | Three.js (r128，CDN 載入) |
| 美術風格 | 麥塊風格（僅使用 BoxGeometry） |
| 物理/碰撞 | 自製空間雜湊網格（格子大小 5） |
| 音效系統 | Web Audio API（全程序化生成，無外部檔案，同時最多 10 個音效） |
| 輸出格式 | 單一 HTML 檔案（內嵌 JS/CSS） |

## 角色系統

### 性別選擇
玩家在主選單選擇男性或女性角色，主選單提示各自的進化路線與所需擊殺數。

| 性別 | 基礎武器 | 攻擊方式 | 進化路線 |
|------|---------|---------|---------|
| 男性 | 劍 + 盾 | 三連擊近戰（斜砍、橫掃、重擊）＋ 劍氣特效 | 劍士 → 賢者 → 大教皇 |
| 女性 | 弓 | 每次射出 5 支扇型追蹤箭矢 | 弓手 → 女賢者 → 大聖母 |

### 進化系統（三階段）

| 階段 | 男性 | 女性 | 解鎖條件 |
|------|------|------|---------|
| 基礎 | 勇者（劍＋盾） | 女勇者（弓） | 遊戲開始 |
| 第二階 | 賢者（紫色斗篷、法杖、隕石雨＋5 顆閃電光球） | 賢者（藍色斗篷、附魔弓、自動 20 支扇型箭矢/秒） | 擊殺 50 隻 |
| 第三階 | 大教皇（白金聖袍、聖杖、光環、自動天雷） | 大聖母（白銀聖袍、天弓、光環、自動 200 支圓周箭矢） | 擊殺 500 隻 |

### 賢者能力
- **男性賢者**：自動隕石雨（冷卻 0.3 秒）＋ 5 顆環繞閃電光球（旋轉速度 5.0，自動攻擊範圍 5）
- **女性賢者**：自動扇型箭雨（20 支箭矢，約 120° 展開，冷卻 1 秒，全追蹤）

### 大教皇 / 大聖母能力
- **大教皇**：天雷從天而降（冷卻 0.0375 秒、每次 5~10 道隨機延遲落雷、偵測範圍 5、AoE 傷害範圍 8、每道雷有隨機位置偏移、巨大雷柱＋地面閃光＋外圈光環）
- **大聖母**：200 支箭矢以 360° 圓周射出（冷卻 1 秒，全部自動追蹤史萊姆）

### 手機升級行為
升級為賢者/大教皇/大聖母後，虛擬搖桿中央按鈕從「ATK」變為綠色「AUTO」，停止手動攻擊功能（因為已自動攻擊）。

## 戰鬥系統

### 連擊系統（男性）
| 段數 | 名稱 | 持續時間 | 攻擊範圍 | 傷害 |
|------|------|---------|---------|------|
| 第一段 | 斜砍 | 0.2 秒 | 3.5 | 1 |
| 第二段 | 橫掃 | 0.18 秒 | 3.5 | 1 |
| 第三段 | 重擊 | 0.3 秒 | 4.5 | 2 |

連擊在 3 秒無操作後重置。

### 箭矢系統（女性）
- 每次射出 5 支扇型箭矢（-0.5 至 +0.5 弧度展開）
- 追蹤轉向速度 8
- 冷卻 0.35 秒、箭速 25、存活時間 3 秒

### 打擊感回饋
- 畫面震動（依攻擊力道縮放）
- 擊中停頓 / 凍結幀（30-80 毫秒）
- 擊中粒子效果（每次命中 8 顆粒子，帶重力）
- 受傷史萊姆白色閃爍
- 擊退物理（帶速度衰減）

## 敵人系統

### 史萊姆（共 1600 隻）
- 80 個群落 × 20 隻，分布在 170 單位範圍
- 所有史萊姆主動追擊玩家（無巡邏模式）
- 1-2 HP，快速擊殺的無雙體驗
- 擊殺後 1 秒重生（含縮小消失動畫）
- 史萊姆之間透過空間雜湊避免重疊
- 無血條顯示（更乾淨的視覺）

### 魔王史萊姆（KING SLIME）
- **觸發條件**：擊殺 3000 隻史萊姆（玩家已是大教皇/大聖母）
- **登場演出**：所有普通史萊姆消失合體，吐司通知 + 全螢幕震動
- **外觀**：原版史萊姆 20 倍大、深紅色身體、紅色眼睛、金色皇冠
- **HP**：5000（需大量攻擊擊敗）
- **行為**：緩慢追擊玩家，接觸造成 10 點傷害（格擋 3 點），附帶擊退
- **血條**：畫面頂部中央紅色血條，即時更新，帶脈動動畫
- **天雷打魔王**：偏移範圍加大（±8），覆蓋巨大身軀
- **箭矢打魔王**：自動追蹤瞄準魔王中心，命中判定範圍配合巨大體型
- **擊敗後**：縮小消失動畫 + 大量粒子爆炸 → 顯示金色「VICTORY」破關畫面
- 魔王戰期間普通史萊姆不會重生

## 操作說明

### PC 操作

| 按鍵 | 功能 |
|------|------|
| W/A/S/D 或 方向鍵 | 移動（速度 9） |
| Space | 跳躍 |
| Shift | 衝刺（速度 25，冷卻 0.8 秒） |
| 滑鼠左鍵 / J | 攻擊 |
| 滑鼠右鍵 / K | 防禦（減少傷害） |
| 滑鼠移動 | 旋轉視角 |
| ESC | 返回主選單 |

### 手機操作（單手）
自動偵測觸控裝置，手機上不使用指標鎖定。

| 操作 | 功能 |
|------|------|
| 底部中央虛擬搖桿（拖曳） | 移動 |
| 點擊搖桿中心（輕觸不拖曳） | 攻擊（升級後自動停用） |
| 螢幕空白區域滑動 | 旋轉視角 |

## 音效系統
使用 Web Audio API 振盪器與噪音緩衝區全程序化生成：
- **主選單音樂**：FF 風格琶音（Am → F → C → G → Am → Em → F → G 進行）
- **戰鬥音樂**：動態戰鬥 BGM
- **音效**：斬擊（3 種連擊變體）、命中、擊殺、衝刺、受傷、死亡、格擋、選單懸停/點擊
- **音效限流**：同時最多 10 個音效播放，避免大量攻擊時音訊爆量

## UI 系統
- 主選單畫面（遊戲標題＋角色選擇＋進化路線提示＋擊殺數提示）
- 遊戲內 HUD：血量條、角色名稱、連擊計數器、KO 總數、史萊姆計數器
- KO 計數器：每 100 擊殺顯示大型中央數字
- 升級通知：吐司風格（從頂部滑入，自動消失）
  - 賢者解鎖：黑底金字主題
  - 大教皇/大聖母解鎖：白色神聖主題
- 魔王血條：畫面頂部中央，紅色脈動動畫，即時更新 HP 數值
- 破關畫面：金色「VICTORY」＋ 總擊殺數統計
- 死亡畫面（重試 / 返回主選單）
- 連擊顯示顏色漸進（白 → 黃 → 橙 → 紅 → 金）
- 左下角即時性能指標面板：FPS、Draw Calls、三角形數、幾何體數、貼圖數、場景物件數、存活史萊姆數、場上箭矢數

## 效能優化
- 空間雜湊網格（格子大小 5）實現 O(1) 碰撞偵測（1600 隻史萊姆）
- 粒子池系統（最多 200 顆粒子）
- 陰影貼圖（2048x2048）
- 霧效深度剔除
- 音效同時播放數上限（10 個），避免 Web Audio API 過載
- 魔王戰期間暫停普通史萊姆更新，減少運算
- 死亡/返回選單時高效清理箭矢
